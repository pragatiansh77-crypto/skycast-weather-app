# SkyCast – Modern Weather Dashboard

A clean, modern, animated weather dashboard built using:

- HTML5
- CSS3
- JavaScript
- Open-Meteo API (Free, No Key Required)

## Features
✔ Current weather  
✔ Hourly Forecast  
✔ 3-Day Forecast  
✔ Animated icons  
✔ Soft white + sky blue modern UI  
✔ Auto-location  
✔ Search city  
✔ Fully responsive  
✔ Single-file project  

## API Used
Open-Meteo (No key required):  
https://open-meteo.com/

## How to Run
Just open the `index.html` file in Chrome.

## Deploy on GitHub Pages
1. Create a new repo  
2. Upload this project  
3. Go to **Settings → Pages**  
4. Select `main` branch → `/root`  
5. Your website will go live!
